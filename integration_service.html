What is an integration service?

An integration service is a secure communication component that transports and can transform data from one organization (business area or partner) to another.

Below is a patterns-based, general purpose integration service:


Essentially, an integration service defines:

  • how applications can exchange messages - example pattern: polling consumer
  • what format will the messages be created - example pattern: event message
  • how will the messages be transmitted - example pattern: publish-and-subscribe
  • what messages should be sent to which system - example pattern: content-based router
  • what message transformation will need to take place - example pattern: content enricher
  • how will the integration be managed and monitored - example pattern: message history
